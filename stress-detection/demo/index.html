How it works

This demo extracts audio signal properties directly in your browser and applies a decision function that mirrors the trained Python model. Upload any short voice clip (3–5 seconds) to see live predictions.

FeatureWhat it measures
MFCCs (40)Vocal tract shape and tonal quality — the most informative feature for speech
delta-MFCCsHow tone changes over time — distinguishes calm (flat) vs angry (rising)
Chroma (12)Pitch class energy — harmonic content of the voice
Spectral Contrast (7)Difference between spectral peaks and valleys
Mel Spectrogram (40)Perceptually-weighted frequency energy
ZCRZero-crossing rate — consonant density and noisiness
RMS energyLoudness — angry speech is loud, sad is soft
For full accuracy, run the Python model: python src/train.py --dataset data/RAVDESS --model svm
The browser demo uses estimated signal properties; the Python model uses all 182 extracted features.
